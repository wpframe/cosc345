◆ MyApp()

MyApp::MyApp ( )

Constructor for the MyApp class.

This constructor initializes the main application, creates a resizable window, sets up an HTML overlay, loads a webpage, registers listeners for various events, and reads stock data from a CSV file.

Create our main App instance.

Create a resizable window by passing by OR'ing our window flags with kWindowFlags_Resizable.

Create our HTML overlay– we don't care about its initial size and position because it'll be calculated when we call OnResize() below.

Force a call to OnResize to perform size/layout of our overlay.

Load a page into our overlay's View

Register our MyApp instance as an AppListener so we can handle the App's OnUpdate event below.

Register our MyApp instance as a WindowListener so we can handle the Window's OnResize event below.

Register our MyApp instance as a LoadListener so we can handle the View's OnFinishLoading and OnDOMReady events below.

Register our MyApp instance as a ViewListener so we can handle the View's OnChangeCursor and OnChangeTitle events below.
