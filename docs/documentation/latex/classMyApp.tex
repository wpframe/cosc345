{\footnotesize\ttfamily My\+App\+::\+My\+App (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Constructor for the \mbox{\hyperlink{classMyApp}{My\+App}} class. 

This constructor initializes the main application, creates a resizable window, sets up an HTML overlay, loads a webpage, registers listeners for various events, and reads stock data from a CSV file. Create our main App instance.

Create a resizable window by passing by OR\textquotesingle{}ing our window flags with k\+Window\+Flags\+\_\+\+Resizable.

Create our HTML overlay-- we don\textquotesingle{}t care about its initial size and position because it\textquotesingle{}ll be calculated when we call \mbox{\hyperlink{classMyApp_a846fdf6a1bf69406024fcad91fe28c9c}{On\+Resize()}} below.

Force a call to On\+Resize to perform size/layout of our overlay.

Load a page into our overlay\textquotesingle{}s View

Register our \mbox{\hyperlink{classMyApp}{My\+App}} instance as an App\+Listener so we can handle the App\textquotesingle{}s On\+Update event below.

Register our \mbox{\hyperlink{classMyApp}{My\+App}} instance as a Window\+Listener so we can handle the Window\textquotesingle{}s On\+Resize event below.

Register our \mbox{\hyperlink{classMyApp}{My\+App}} instance as a Load\+Listener so we can handle the View\textquotesingle{}s On\+Finish\+Loading and On\+DOMReady events below.

Register our \mbox{\hyperlink{classMyApp}{My\+App}} instance as a View\+Listener so we can handle the View\textquotesingle{}s On\+Change\+Cursor and On\+Change\+Title events below.
